The Essentials of Dev Tools

Google

To start out, Google is your best friend as a new OR senior developer. Google is widely as a resource to assist you writing code and finding solutions. Never be afraid to Google the answer if you are stuck. You may find better way of writing out your code or find a new resource to use!

Google Fonts

Of course Google Fonts is a staple for programmers. This database allows you to browse through thousands of fonts to put on your custom websites and apps! You can easily link or download fonts from this database that google provides.

Github

Hopefully you have heard of Github. This website is a great place to put your code out there and very useful when searching for your first programming career. Github allows your to post the code you create on their website for friends and followers to look at(One of the best features of this website) because you can have more expierenced programmers browse your code for errors and offer corrections or solutions.

Netlify

Netlify is a cloud computing company that offers website hosting and serverless backend services. I use this website personally to deploy sites including this one! Setting up and account is quite easy and can be linked right to your Github to easily update files for the websites you launch! The free version gives you limited storage space for websites while the p[aid version allows for more storage and team coordination.

Resources

Stackoverflow

StackOverflow is a question and answer website that allows free communication between developers. There are a WIDE variety of topics when it comes to computer programming. Setting up an account is very easy and allows you to get exactly the help you need from people in the field.

W3Schools

W3Schools is another resource website I found very useful while beginning developing. It is a resource for a number of programming languages. They have exersizes and quizzes you can take to practice your programming skills and offer certificaions for passing their programming tests. I like this website for providing a simple library where you can look up basic and some more advanced coding examples and guides.

Design

Figma

Figma is a design website for laying out the content of your website. Figma you a visual sample of what your website will look like that you can show off to your friends or coworkers!

Sketch

Sketch is similar to Figma. Sketch descripes itself as a 'digital design toolkit.'. Personally I have used Figma more often than Sketch. This is your personal prefernce and what you are comfortable with. It's always nice to have a outline of what you are creating!

Gimp

Gimp is a FREE clone of Adobe Photoshop. You can use this downloadable desktop app to edit and resize images. The subscriptions to libraries and websites can start to add up when you are beginning developing its always nice to have a free option!

Copyright Free Media

Unsplash

Unsplash is a copyright free media website. The website claims to have over 200,000 contributing photographers and generates billions of photo impressions a year. There are over 2 MILLION photos on this website that can be sorted by theme, color scheme and more.

Pexel

Pexels is a similar website to Unsplash. It also has millions of copyright free photos and VIDEOS that can easily be searched by whatever theme you are creating.

Fontawesome

Fontawesome is a online library with thousands of different icons. While there is a paid subsription some icons remain free for all. You can directly and easily link this website to your HTML file and use the icons with ease. Icons can be searched easily while the depth is limited for the free verisons.

There are many resources out there! Please share if you think one NEEDS to be on this website!
